Personalized Yoga Therapy Management System

Evidence based database based on IAYT module

Purpose

This system is designed to manage and organize yoga therapy practices based on evidence from research papers and clinical studies. It enables researchers and practitioners to create structured therapy modules for various diseases and conditions, track supporting evidence through Randomized Controlled Trials (RCTs), and generate personalized practice recommendations based on scientific data.

System Logic & How It Works

1. Module-Based Structure

The system is organized around Modules, where each module represents a research paper or study that focuses on yoga therapy for a specific disease. A single disease can have multiple modules, as different research papers may propose different sets of practices for the same condition.

For example, Depression may have Module A (developed by Naveen et al., 2013) with one set of practices, and Module B (developed by another research team) with a different set of practices. Each module is linked to its source research paper through a hyperlink.

2. Practices Within Modules

Each module contains multiple Yoga Practices organized across nine practice segments: Preparatory Practice, Breathing Practice, Suryanamaskara, Yogasana, Pranayama, Meditation, Additional Practices, Kriya (Cleansing Techniques), and Yogic Counselling. Practices are automatically associated with the disease that the module addresses.

3. RCT Database & Evidence Scoring

The system maintains a comprehensive RCT (Randomized Controlled Trial) Database that tracks clinical studies and research papers supporting specific practice-disease combinations. Each RCT entry includes detailed information about the study, participants, interventions, and results.

RCT Score: The system calculates an RCT count for each practice within a disease context. The higher the RCT score (number of supporting research studies), the stronger the evidence for that practice in treating that particular disease. This scoring mechanism helps prioritize practices with stronger scientific backing when generating recommendations.

4. Recommendation Generation

When generating therapy recommendations, the system combines practices from relevant modules, removes duplicates, applies contraindications based on disease combinations, and prioritizes practices with higher RCT scores. This ensures that recommendations are evidence-based, safe, and personalized to the specific disease profile.
